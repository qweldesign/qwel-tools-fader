# QWEL Fader Gallery

- フェードアニメーションで遷移するギャラリー
- **使い方**:  
  _fader.scss をバンドルした css を読み込み,  
  ギャラリー本体に [data-gallery="fader"] 属性を付与し,  
  ギャラリーアイテムに [data-gallery-item] 属性を付与する
- **オプション**  
  data-interval: アニメーション時間間隔 (data属性で指定)

▶ Sample DEMO: [https://qwel.design/tools/fader/]
